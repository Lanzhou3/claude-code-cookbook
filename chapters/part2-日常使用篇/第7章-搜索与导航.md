## 7.2 文件搜索 (Glob)

### 7.2.1 基本模式

```bash
# 通配符搜索
> 找所有 .ts 文件
> 找所有 .tsx 文件
> 找所有 .css 文件

# 目录搜索
> 查找 src/components 下的所有文件
> 查找 tests 目录中的测试文件

# 递归搜索
> 查找所有 package.json 文件
> 查找所有 node_modules 目录
```

### 7.2.2 高级模式

```bash
# 排除目录
> 找所有 .ts 但排除 node_modules
> 找所有 .tsx 但排除 __tests__

# 嵌套搜索
> 查找 src/**/index.ts
> 查找 **/components/**/*.tsx

# 多个模式
> 找所有 .ts 和 .tsx 文件
```

### 7.2.3 Glob 选项

| 选项 | 说明 | 示例 |
|------|------|------|
| `-d` | 只返回目录 | `Glob -d **/components` |
| `-f` | 只返回文件 | `Glob -f **/*.ts` |
| `-n` | 显示行数 | `Glob -n **/src/**` |

## 7.3 内容搜索 (Grep)

### 7.3.1 基本搜索

```bash
# 搜索关键词
> 在 src 目录搜索 "useState"
> 搜索 "async function"
> 搜索 "console.log"

# 显示行号
> 搜索并显示行号 -n "export"
```

### 7.3.2 正则表达式

```bash
# 精确匹配
> grep "handle\w+" src/

# 函数定义
> grep "^function\s+" src/

# 字符串字面量
> grep "\"https://" src/
```

### 7.3.3 搜索上下文

```bash
# 显示匹配前后几行
> 搜索 "useEffect" -B 2 -A 3

# 显示文件上下文
> 搜索 "TODO" -c 5

# 只显示匹配的文件
> 搜索 "FIXME" -l
```

### 7.3.4 Grep 选项

| 选项 | 说明 | 示例 |
|------|------|------|
| `-i` | 忽略大小写 | `grep -i "error"` |
| `-r` | 递归搜索 | `grep -r "func"` |
| `-l` | 只显示文件 | `grep -l "class"` |
| `-n` | 显示行号 | `grep -n "fn"` |
| `-c` | 显示计数 | `grep -c "if"` |
| `-B N` | 之前 N 行 | `grep -B 2 "err"` |
| `-A N` | 之后 N 行 | `grep -A 3 "err"` |

## 7.10 本章小结

| 功能 | 工具/命令 | 说明 |

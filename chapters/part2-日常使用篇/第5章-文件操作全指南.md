## 5.1 文件读取

### 5.1.1 基本读取

```bash
# 读取单个文件
> 读取 src/index.ts 的内容

# 读取多个文件
> 同时查看 src/a.ts 和 src/b.ts

# 读取并高亮特定行
> 查看 src/main.ts 的第 10-20 行
```

### 5.1.2 读取选项

| 选项 | 说明 | 示例 |
|------|------|------|
| `-n` | 显示行号 | `读取 -n src/file.ts` |
| `-c N` | 显示上下文 | `读取 -c 5 src/file.ts` |
| `--encoding` | 指定编码 | `读取 --encoding utf-8` |

### 5.1.3 大文件处理

对于大文件，Claude Code 会智能处理：

```
> 读取一个 10MB 的日志文件

文件较大（10.2MB），自动采用以下策略：
- 显示文件前 100 行
- 使用 tail 查看最新内容
- 可使用 /grep 搜索特定内容
```

## 5.3 文件编辑

```bash

> 删除第 20-25 行
```

### 5.3.2 块编辑

```bash
# 在函数内添加代码
> 在 `calculate()` 函数中添加参数验证

# 在文件末尾添加
> 在文件末尾添加新的导出

# 包裹代码
> 把这五行代码用 try-catch 包裹
```

### 5.3.3 编辑示例

**添加内容：**
```
原文件：
1 | export function add(a, b) {
2 |   return a + b
3 | }

编辑：在这行后添加 `  if (typeof a !== 'number') throw new Error('a must be number')`

编辑后：
1 | export function add(a, b) {
2 |   if (typeof a !== 'number') throw new Error('a must be number')
3 |   return a + b
4 | }
```

**替换内容：**
```
原文件：console.log('debug')

编辑：把 'debug' 改为 'info'

编辑后：console.log('info')
```

## 5.7 权限与安全

### 5.7.1 受保护路径

Claude Code 默认保护以下路径：

| 路径 | 保护原因 |
|------|----------|
| `/etc/*` | 系统配置文件 |
| `~/.ssh/*` | SSH 密钥 |
| `~/.aws/*` | 云服务凭证 |
| `**/.env` | 环境变量文件 |
| `**/secrets.*` | 密钥文件 |

### 5.7.2 操作限制

```
> 读取 /etc/passwd

⛔  拒绝访问：系统配置文件受保护

> 编辑 ~/.ssh/id_rsa

⛔  拒绝访问：SSH 密钥文件受保护
```

### 5.7.3 自定义保护

你可以在配置中添加自定义保护路径：

```json
{
  "permissions": {
    "protectedPaths": [
      "**/credentials.json",
      "**/config.production.json"
    ]
  }
}
```
